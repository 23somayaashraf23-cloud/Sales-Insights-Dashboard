# 📊 Sales Analysis Dashboard (Excel)

## 📌 Overview

This project presents an **interactive Sales Analysis Dashboard built entirely in Microsoft Excel**.
The dashboard transforms raw sales data into meaningful insights to support better business decision-making.

It provides a comprehensive view of **sales performance, profitability, customer distribution, and product performance**.

---

## 🎯 Objectives

* Analyze overall sales and profit performance
* Identify top and bottom performing products
* Understand category and subcategory trends
* Evaluate the impact of discounts on profit
* Explore customer distribution across regions

---

## 📊 Dashboard Features

### 🔹 Key Metrics

* Total Sales: **$9.8M+**
* Total Quantity Sold: **37K+**
* Total Profit: **$1.43M**
* Average Discount: **16%**
* Profit Margin: **14%**

### 🔹 Visual Insights

* **Sales by Category** (Technology, Furniture, Office Supplies)
* **Quantity by Category**
* **Profit by Category**
* **Profit & Sales by Subcategory**
* **Top 10 Products by Sales**
* **Bottom 10 Products by Sales**
* **Customer Distribution by State (Map Visualization)**

### 🔹 Interactivity

* Dynamic filters (Slicers) for:

  * Year (2015–2018)
  * Category
  * Quarter (Q1–Q4)

## 📁 Dataset

The dataset includes:

* Orders information
* Sales, Profit, and Quantity
* Product categories & subcategories
* Customer locations

*(Dataset can be replaced with any similar structured sales dataset.)*

---

## 💡 Key Insights

* Technology category generates the highest profit
* Office Supplies contribute the highest quantity sold
* Some products generate high sales but low profit (due to discounts)
* Significant variation in customer distribution across states

---

## 🚀 How to Use

1. Download the Excel file from this repository
2. Open using Microsoft Excel (recommended version: 2016 or later)
3. Use slicers to filter data dynamically
4. Explore different visuals for insights
